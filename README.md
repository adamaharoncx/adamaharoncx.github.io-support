# WishSync Support

Need help with **WishSync**? This page is the official support resource for the app.

## Contact

**Email:** [WishSync@engineer.com](mailto:WishSync@engineer.com)

## What we can help with

- Account access issues  
- Partner connection problems  
- Wishlist syncing issues  
- App bugs or unexpected behavior  
- Feedback and feature requests  
- General support questions  

## Response Time

We aim to respond as soon as reasonably possible.

## About WishSync

WishSync helps couples keep meaningful things organized in one place, including:

- Personal wishlists  
- Shared gift ideas  
- Important dates like birthdays and anniversaries  
- Private gift planning  
- Notes, links, and saved inspiration
